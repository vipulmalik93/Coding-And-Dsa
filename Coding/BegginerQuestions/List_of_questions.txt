Here are 20 very basic coding questions for a complete beginner:

Basic Output & Input

Write a program to take a user's name as input and print "Hello, [name]!".
Write a program that takes two numbers as input and prints their sum.

Basic Arithmetic & Operators

Write a program to calculate the area of a rectangle (length × width).
Write a program to swap two numbers without using a third variable.
Write a program to check if a number is even or odd.
Write a program to find the largest of three numbers.
Write a program to calculate the square and cube of a number.

Conditions & Loops

Write a program to print numbers from 1 to 10 using a loop.
Write a program to print the multiplication table of a given number.
Write a program to find the factorial of a number using a loop.
Write a program to count the number of digits in a given number.

Arrays & Strings

Write a program to find the largest number in an array.
Write a program to reverse a given string.
Write a program to check if a string is a palindrome (same forwards and backwards).

Functions

Write a function to check if a number is prime.
Write a function to return the sum of all numbers in an array.
Write a function that takes two numbers as arguments and returns their greatest common divisor (GCD).

Miscellaneous
Write a program to print the Fibonacci series up to n terms.
Write a program to check if a number is an Armstrong number (e.g., 153 = 1³ + 5³ + 3³).
